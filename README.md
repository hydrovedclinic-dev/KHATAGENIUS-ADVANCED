# KhataGenius (WhatsApp + PDF)

A fast mobile-first khata entry app with:
- Shop + item auto-learning
- Qty remember
- WhatsApp share formats
- PDF export
- Summary filters (Daily / Weekly / Monthly / Yearly / Date range)

## Run locally
Just open `index.html` in Chrome.

## Host on GitHub Pages
1. Upload all files in this repo
2. GitHub → Settings → Pages
3. Source: Deploy from branch → `main` → `/root`
4. Open the GitHub Pages URL

## PWA install
Open the site in Chrome → **Add to Home Screen**
