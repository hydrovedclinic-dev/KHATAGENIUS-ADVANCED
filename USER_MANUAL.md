# KhataGenius — User Manual (Easy Guide)

KhataGenius helps you quickly record purchases (Cash / Credit), keep shop-wise balances, and share details on WhatsApp.
It also generates PDF reports and Summary reports with filters.

---

## 1) App Screens (What you will see)
### ✅ A) Entry Screen (Main)
This is the main screen where you enter:
- **Shop Name**
- **Item Name**
- **Qty**
- **Rate**
- **Mode:** Cash / Credit
- **Date & Time** (auto captured)

### ✅ B) Records / History Screen
Shows all saved entries with search and filter.

### ✅ C) Shops Screen
Shows shop-wise totals and pending due.

### ✅ D) Summary Screen (New)
Advanced summary with filters:
✅ Daily | Weekly | Monthly | Yearly | Date Range

---

## 2) How to Add a New Entry
1. Open the app
2. Select/Type **Shop Name**
3. Select/Type **Item Name**
4. Enter **Qty**
5. Enter **Rate**
6. Mode:
   - Select **CASH** if payment done
   - Select **CREDIT** if due
7. Tap **SAVE**

✅ Entry is saved instantly with date & time.

---

## 3) Smart Auto Learning (Very Fast Entry)
KhataGenius remembers:
✅ Recently used **Shop Names**  
✅ Recently used **Item Names**  
✅ Last used **Qty** for item (Qty remember)  
✅ Last entered **Item Price** for the same shop

So next time, entry becomes almost 1-click.

---

## 4) WhatsApp Share (Billing / Due Message)
You can share:
✅ Individual purchase details  
✅ Shop-wise pending amount  
✅ Filtered Summary (Daily/Weekly/Monthly/etc.)

### How to share
1. Open the record / shop
2. Tap **WhatsApp**
3. Select contact in WhatsApp and send

---

## 5) PDF Report (Print Ready)
You can generate PDF:
✅ Shop bill PDF  
✅ Summary PDF (Filtered)

### PDF includes
- Shop name
- All items with qty, rate, amount
- Total amount
- Date range (for summary PDF)

✅ Best for printing or keeping records.

---

## 6) Summary Filters (Daily/Weekly/Monthly/Yearly/Date Range)
Go to **Summary**
Choose:
- **Daily** → today’s entries
- **Weekly** → Mon to Sun
- **Monthly** → current month
- **Yearly** → current year
- **Date Range** → choose From & To date

Then click **Apply Filter**

✅ Summary will show shop-wise:
- Credit
- Cash
- Due

---

## 7) Reset Summary Filter
In Summary screen:
- Tap **Reset**
✅ Back to Daily (default)

---

## 8) Settle Payment (Clear Due)
When the shop owner pays you (or you pay them):
1. Open shop
2. Tap **Settle / Paid**
3. Enter amount or confirm

✅ Due will reduce/clear.
✅ You can send settlement message on WhatsApp.

---

## 9) Backup & Restore (Very Important)
Your data is stored in phone browser storage.
To prevent data loss:

### ✅ Backup
1. Open menu/settings (if available)
2. Tap **Backup**
3. Save backup file

### ✅ Restore
1. Open menu/settings
2. Tap **Restore**
3. Select your backup file

⚠️ Recommendation: Take backup weekly.

---

## 10) Install as Mobile App (PWA)
If hosted on GitHub Pages:
1. Open the app URL in **Chrome**
2. Tap **⋮ Menu**
3. Tap **Add to Home screen**
4. Install

✅ It will behave like a real mobile app
✅ Opens full screen, faster loading

---

## 11) Best Usage Tips
✅ Make separate shop name for each vendor (Example: “D-Mart”, “Medical Store”, “Milk Man”)  
✅ Always use correct mode (Cash/Credit)  
✅ Use Summary filter every month to review spending  
✅ Export PDF whenever needed for proof

---

## 12) Troubleshooting
### App not saving?
- Ensure you pressed **Save**
- Don’t use private/incognito mode

### Dropdown not showing?
- Start typing 2–3 letters
- Previously saved names will appear

### WhatsApp not opening?
- Check if WhatsApp is installed
- Try again after giving permission

### PDF not downloading?
- Allow browser downloads
- Try Chrome browser
